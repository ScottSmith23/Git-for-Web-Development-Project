Stretch Questions

    1. What is the difference between an inline element and a block element?
		-An inline element does not cause a line break while a block element always starts on a new line and takes up the full width of the page. Inline elements like <span> are used as a container for HTML text and can be used to style a certain text element withing a larger text while block elements like <div> are used to separate some HTML elements from the rest, the <div> element is unstyle so it does not change the look of an HTML element.
    2. What happens when an element is positioned absolutely? 
	- When an element is position absolutely it is position automatically to the starting point or top-left corner of it's parent element, it will remove the element from the document flow and other elements behave as if the absolute element is removed completely from the webpage
    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
	- You can use display inline because the width properties can be changed
    4. Name 3 elements that are display block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
		- <header>,<div>,<p> are display block. <span>,<a> are inline by default. The <img> element is inline-block by default
    5. In your own words, explain the box model. What is the fix for the box model? 
		-The box model is a box that wraps around every HTML element and consists of a Margin,Border,Padding and the Content itself respectively.
